# 2026-06-24 PATCS 3D 概率云阶段记录

## 当前结论

当前 3D 概率云团仍然无法形成预期中的“顺滑外形 + 清晰节点限制范围”的最终形态。

已确认的问题：

- `phase_2d` 模式本质是逐 phase 的 2D normal-plane Gaussian 截面拼接，容易产生拼接感和局部扭转。
- `global_3d_gaussian` 模式已经能显示一次性 3D density cloud，但目前只是大概率团可视化，尚未真正把 demo_i 条件收束场作用到连续 3D density 上形成最终变形 isosurface。
- 当前 olive / precision channel / black-hole 逻辑仍主要体现在半径曲线和局部截面诊断上，外形还不够顺滑，节点限制范围也不够清晰。
- 后续不能只调可视化参数，需要联合调整概率云构建逻辑、收束场和训练优化器/loss 权重。

下一阶段目标：

```text
global 3D density
-> demo_i-conditioned contraction field
-> olive envelope
-> narrow precision channel
-> final black-hole anchor
-> smooth isosurface / differentiable loss
```

### Supervision Schema

当前设计方向：

```text
一个原始监督条
= xyz 概率团
+ pose 概率团
+ gripper 硬监督
```

- xyz：轨迹/空间路径概率团。
- pose：姿态/末端位姿概率团，独立计算。
- gripper：硬监督，同时作为 xyz/pose 两个概率团的状态条件。

真实 pose 接入时应使用 rotation vector / axis-angle / quaternion log map，不能直接把 raw quaternion 当普通欧氏向量。
